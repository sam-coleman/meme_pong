# User Instructions     


## Dependencies
* Install pygame
* Install numpy

## Installing
Install pygame
```bash
sudo apt-get install python3-pygame
```
Install numpy
```bash
sudo apt-get install python3-numpy
```

## Executing Program
```bash
python3 pongV3.py
```
